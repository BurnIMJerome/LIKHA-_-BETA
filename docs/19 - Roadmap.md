# Roadmap

This roadmap is a working placeholder and should be adjusted as product priorities change.

## Near Term

- Production hardening and expanded audit views for agent guardrails, delegation, approvals, and embedded chat.
- Improve agent evaluation, task-history filtering, and structured outcome reporting.
- Harden designer run versus unattended job behavior.
- Improve browser and desktop selector reliability.
- Expand event trigger coverage.
- Improve robot job visibility.
- Add more focused regression tests.
- Improve documentation coverage.

## Mid Term

- Reusable agent and tool templates.
- Agent evaluation, test scenarios, and outcome scoring.
- Central policy administration for distributed agent and robot deployments.
- Package deployment to robot VMs.
- File artifact upload from robot runs.
- More detailed robot assignment rules.
- Additional credential-vault integrations beyond the current SecureString and agent redaction controls.
- Additional AI document extraction activities.
- More Control Room administration pages.

## Longer Term

- Multi-user roles and permissions.
- Database migration and export tools for moving records between SQLite, PostgreSQL, and Microsoft SQL Server.
- High availability Control Room patterns.
- Advanced queue analytics.
- Marketplace or reusable component library.
