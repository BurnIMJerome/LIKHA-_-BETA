# Convert Text to Date Time

![Likha Process Designer](../../images/process-designer.png)

**Activity group:** Data

## Purpose

Converts text into a DateTime variable.

## Properties

- `input`: Text value or variable name to convert.
- `custom_format`: Optional date/time format used to parse the text.
- `output`: DateTime variable to store the result.

Common format tokens include `yyyy`, `yy`, `MM`, `dd`, `HH`, `hh`, `mm`, `ss`, and `tt`.
