# Wait

![Likha Process Designer](../../images/process-designer.png)

**Activity group:** Flow control

## Purpose

Pauses the flow for a fixed number of seconds.

## Properties

- `seconds`: Number of seconds to wait.

## Example

```txt
seconds: 2
```
